# Indian Card Games Hub

Indian Card Games Hub is a calm educational collection of notes about shared ideas across Indian card games. The goal is to make table reading, hand judgment, timing, and repeated mistakes easier to understand for readers who want practical learning material instead of noisy strategy slogans.

## What This Repository Covers

This repository focuses on the habits that carry across many card games: reading the flow of a round, protecting hand value, recognizing common player errors, and making better decisions when information is incomplete.

The articles are written for beginners who want clear explanations, but they are also structured to remain useful for intermediate readers who are trying to review their games more carefully.
